Key Definitions for Special Functions and Characters
Key
 Combination Control Function or Character
 An Alternate key-Sets the high order bit on a
 character. Press ALT char.
 CTRL Use as a control key.
 BREAK or CTRL ~ Stops the program currently executing.
 CTRL El Generates an underscore (_).
 CTRL ~ Generates a left brace ({).
 CTRL ~ Generates a right brace (}).
 CTRL ~ Generates a tilde (-) character.
 CTRL ID Generates a reverse slash ( \ ).
 CTRL BREAK Generates an end-of-file (EOF). This
 sequence is the same as pressing Esc on a
 standard terminal.
 f:i:l or cTR~ 0 Generates a backspace.
 sHm [~E] or Deletes the entire current line.
 CTRL CE
 SHIFT BREAK OT' Interrupts the video display of a running
 CTRL ~c program. This sequence reactivates the
 shell and then runs the program as a back
 ground task.
 CTRL ~ Upper-/lowercase shift lock function.
 CTRL ~ Generates a vertical bar (1) in reverse video.
 CTRL CD Generates an up arrow or caret (").
 CTRL CE Generates a left bracket (U.
 CTRL s0 Generates a right bracket (]).
