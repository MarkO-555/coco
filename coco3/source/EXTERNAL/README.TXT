The files in this directory rebuild EXTERNAL.EXE.  EXTERNAL.EXE is a patch
program which adds extra features to CoCo BASIC when used in conjunction with
the CoCo 3 Emulator version 1.6 or later.  EXTERNAL.EXE can be downloaded from 

http://www.vavasour.ca/jeff/trs80.html#coco3

BUILD5.BAT	Contains commands to rebuild EXTERNAL.EXE under Microsoft's
		Macro Assembler (MASM) version 5.x.
BUILD6.BAT	Contains commands to rebuild EXTERNAL.EXE under Microsoft's
		Macro Assembler (MASM) version 6.x.
